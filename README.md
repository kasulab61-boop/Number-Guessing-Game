# 🎮 Number Guessing Game

This is a simple Python-based number guessing game.

## 🔹 Features
- Random number generation
- User input handling
- Provides hints (higher/lower)

## 🔹 How to Run
1. Install Python
2. Run the file:
   python project.py

## 🔹 Description
The program generates a random number and asks the user to guess it. 
Based on the input, it gives hints until the correct number is guessed.
